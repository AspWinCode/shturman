## 🔐 Шаг 1 — Войти в Expo аккаунт (1 минута)

```bash
eas login
# Введите email и пароль от expo.dev
```

---

## 🔧 Шаг 2 — Инициализировать EAS проект (1 минута)

```bash
cd D:/travel_app
eas init
# Выбери: "Create a new EAS project"
# Название: shturman
```

После этого в `app.json` автоматически пропишется `extra.eas.projectId`.

---

## 🏗️ Шаг 3 — Собрать Preview APK для тестирования

```bash
eas build --platform android --profile preview
```

- Сборка занимает ~10-15 минут на серверах Expo
- Ссылка на APK придёт в терминал и на email
- Скачай APK и установи на устройство для проверки

---

## 🚀 Шаг 4 — Собрать Production AAB для RuStore

```bash
eas build --platform android --profile production
```

- Результат: `.aab` файл (Android App Bundle)
- Ссылка будет в терминале — скачай его

---

## 📦 Шаг 5 — Загрузить в RuStore Console

1. Открой https://console.rustore.ru
2. Перейди в своё приложение → **Версии** → **Создать версию**
3. Загрузи `.aab` файл
4. Заполни:
   - **Описание** (ru): см. ниже
   - **Скриншоты**: минимум 4 штуки (1080×1920)
   - **Иконка**: `assets/store-icon.png` (512×512)
   - **Feature graphic**: `assets/feature-graphic.png` (1024×500)
   - **Категория**: Путешествия
   - **Возрастной рейтинг**: 6+

## ⚡ Быстрые команды

```bash
# Проверить TypeScript
npx tsc --noEmit

# Пересобрать иконки
node scripts/generate-assets.js
node scripts/generate-feature-graphic.js

# Preview сборка (APK для тестирования)
eas build --platform android --profile preview

# Production сборка (AAB для RuStore)
eas build --platform android --profile production

# Посмотреть статус сборок
eas build:list
```
