## App name
Штурман

## Short description (RU)
Планируйте поездки с маршрутами, транспортом, отелями и бюджетом в одном приложении.

## Full description (RU)
Штурман помогает собрать поездку целиком: выбрать даты и бюджет, подобрать отель и транспорт, получить маршрут по дням и сохранить документы в Travel Wallet. Приложение поддерживает fallback-сценарии при недоступности live API и подходит для планирования как коротких, так и длинных поездок.

## Short description (EN)
Plan trips end-to-end with routes, transport, hotels, and budget tracking.

## Full description (EN)
Shturman helps you build a complete trip plan: dates, budget, hotels, transport options, day-by-day itinerary, and travel documents in one place. The app includes resilient fallback UX when live providers are unavailable.
