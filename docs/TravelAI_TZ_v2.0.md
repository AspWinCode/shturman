## 1. Введение и цели проекта

### 1.1. Описание продукта

**Штурман** — мобильное приложение для самостоятельных путешественников, которое автоматически составляет персонализированный маршрут поездки по России и миру на основе бюджета, дат и интересов пользователя. Приложение агрегирует транспорт (Яндекс Путешествия), жильё (Островок), достопримечательности и готовые маршруты в единую экосистему.

### 1.2. Ключевые изменения в v2.0

| Компонент | v1.0 | v2.0 |
|-----------|------|------|
| Транспорт | Aviasales API + deep links | **Яндекс Путешествия** deep links |
| Жильё | Hotellook API | **Островок** deep links |
| Авторизация | Email + пароль | **Яндекс ID / Google** + Email |
| Карта | Не реализована | **Google Maps / Яндекс Карты** |
| Погода | Не реализована | **OpenWeatherMap API** |
| Travel Wallet | Не реализован | **Загрузка билетов и броней** |
| Оффлайн | Не реализован | **Полный оффлайн-режим** |
| Доступность | NFR-заглушка | **Полный модуль ♿** |
| Тёмная тема | Не реализована | **Dark Mode** |

### 1.3. Бизнес-цели

- Создать AI-планировщик путешествий, сокращающий время планирования до 1–2 минут
- Монетизировать через комиссии партнёров и Premium-подписку
- Достичь MAU 50 000+ за первые 12 месяцев
- Стать первым российским travel-приложением с полноценным модулем доступности

## 3. Ценностное предложение

> **"Спланируй идеальное путешествие за 60 секунд"** — введи параметры и получи готовый маршрут с транспортом, жильём и расписанием по дням.

### 3.1. Конкурентные преимущества

- Единое место: маршрут + транспорт + жильё + активности
- Глубокая интеграция с российскими сервисами (Яндекс, Островок)
- Первый travel-планировщик с модулем доступности для инвалидов
- Оффлайн-режим: всё работает без интернета во время поездки
- Travel Wallet: документы, билеты, брони — в одном месте

## 4. Архитектура приложения

### 4.1. Клиентская часть

- **Мобильное приложение** iOS / Android — React Native (Expo SDK 51+)
- **Навигация** — Expo Router (file-based)
- **Состояние** — Zustand + SQLite (expo-sqlite) для персистентности
- **Административная панель** — React.js (веб, отдельный репозиторий)
- **Партнёрский кабинет** — веб-приложение

### 4.2. Серверная часть

- REST API (Node.js / FastAPI)
- PostgreSQL (основная БД) + Redis (кэш) + Elasticsearch (поиск мест)
- AI-сервис: OpenAI GPT API (Production) / шаблонная генерация (MVP)
- Push: Firebase Cloud Messaging (FCM) / APNs
- Хранение файлов: AWS S3 / Cloudflare R2
- CDN для медиа

### 4.3. Внешние интеграции (v2.0)

| Интеграция | Назначение | Приоритет |
|-----------|-----------|-----------|
| **Яндекс Путешествия** | Поиск и бронирование транспорта (deep links) | Must Have |
| **Островок (ostrovok.ru)** | Поиск и бронирование отелей (deep links) | Must Have |
| **Яндекс ID / Google OAuth** | Авторизация | Must Have |
| **Google Maps / Яндекс Карты** | Карта маршрута, геокодинг | Must Have |
| **OpenWeatherMap API** | Прогноз погоды на даты поездки | Should Have |
| **Firebase FCM** | Push-уведомления | Must Have |
| **OpenAI GPT API** | AI-генерация маршрутов (Production) | Should Have |
| **Sentry** | Мониторинг ошибок | Must Have |
| **Firebase Analytics + Amplitude** | Продуктовая аналитика | Must Have |
| **SendGrid / Brevo** | Транзакционные email | Must Have |
| **Stripe / ЮKassa** | Premium-подписка | Should Have |
