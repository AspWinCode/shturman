# Assets — требования для RuStore

Все файлы в этой папке должны быть созданы перед сборкой.
Используйте Figma, Adobe Illustrator или любой редактор.

---

## Обязательные файлы для сборки

| Файл | Размер | Формат | Назначение |
|------|--------|--------|------------|
| `icon.png` | 1024×1024 | PNG, без прозрачности | Иконка приложения (iOS + Android) |
| `adaptive-icon.png` | 1024×1024 | PNG, с прозрачностью | Адаптивная иконка Android (foreground) |
| `splash.png` | 1284×2778 | PNG | Сплэш-скрин |

### Цвета бренда
- Основной (фон адаптивной иконки): `#1B3A6B` (navy)
- Акцент: `#E8472B` (красный)
- Фон сплэша: `#1B3A6B`

### Рекомендации по иконке
- Логотип «штурвал» или компас на тёмно-синем фоне
- Без текста (плохо читается в малом размере)
- Отступы от краёв: минимум 10% с каждой стороны

---

## Скриншоты для RuStore (обязательно минимум 4)

RuStore принимает PNG/JPG, размер: **1080×1920** (портрет) или **1920×1080** (ландшафт).

| № | Экран | Что показать |
|---|-------|-------------|
| 1 | Главный экран | Лента поездок, популярные города |
| 2 | Создание поездки | Форма с городами и датами |
| 3 | Результат — Маршрут | День 1–3 с карточками мест |
| 4 | Транспорт | Aviasales live цены + кнопка купить |
| 5 | Бюджет | Диаграмма расходов |
| 6 | Discover | Каталог маршрутов с фильтрами |

### Снятие скриншотов
```bash
# Запустить приложение в Expo Go или сборке
npx expo start

# Снять скриншот на устройстве или эмуляторе
# Android: кнопка питания + громкость вниз
# Эмулятор Android Studio: кнопка camera в боковой панели
```

## Быстрый способ создать placeholder-иконку

Если нет дизайнера — можно использовать:
- https://icon.kitchen — генератор иконок для Android/iOS
- https://appicon.co — конвертер из одного PNG во все нужные размеры
- https://www.figma.com/community/file/1114112476261038859 — шаблон

Минимальный вариант: синий квадрат #1B3A6B + белый штурвал/компас.
